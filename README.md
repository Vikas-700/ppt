# Tejas 2.0: Personality Development Event

This repository contains resources, presentations, and insights from the **Tejas 2.0** event, a collaborative initiative organized by **Infinity Personality Development** in association with **Invertis University**. The event focused on enhancing personal growth and professional skills, offering workshops and sessions on various aspects of personality development.

## Event Overview:
Tejas 2.0 aimed to empower students and professionals through engaging activities that fostered self-awareness, communication, leadership, and interpersonal skills. As the team leader, I contributed to the event's success by coordinating various segments and ensuring smooth execution of activities.
